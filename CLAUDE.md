QUI ĐỊNH VỀ VIỆC NỘP BÀI
- Đầu mỗi file mã nguồn phải có các dòng comment thể hiện thông tin cụ thể của
sinh viên như sau:

    // Mã số sinh viên: 25730098
    // Họ và tên: Phạm Nguyễn Tuấn Anh
    // Ngày sinh: 15/05/1996
    // Lớp: CN1.K2025.2.CNTT

- Tất cả các bài tập được đặt bên trong một thư mục, tên thư mục theo quy định
như sau: BTTH02_MSSV_HoVaTen. Ví dụ Sinh viên Nguyễn Sơn Trà có
MSSV là 24521269 thì đặt tên thư mục như sau: BTTH02_24521269_
NguyenSonTra
- Sau đó nén thư mục trên thành tập tin .zip hoặc .rar (tên file nén cũng theo quy
định như tên thư mục). Ví dụ BTTH02_24521269_ NguyenSonTra.rar
